# js-html-css

Коллекция практических примеров на JavaScript, HTML и CSS.

## О проекте

Репозиторий содержит два примера:

### reaction — Тест на реакцию
Игра, которая тестирует вашу скорость реакции. Нажмите кнопку "Start", ждите зеленого круга и кликайте как можно быстрее.

**Как играть:**
1. Нажмите кнопку Start
2. Ждите случайное время (1-4 секунды)
3. Как только круг станет зеленым, кликайте по нему
4. Система покажет время вашей реакции

**Функции:**
- Случайная задержка через setTimeout
- Измерение времени реакции
- Отображение результата
- Предотвращение ранних кликов

### snowflakes — Анимация снегопада
Демонстрация CSS анимаций и JavaScript для создания динамических элементов.

**Функции:**
- 200 падающих снежинок
- Разные размеры и скорости
- Градиентный фон
- Анимация через CSS keyframes

## Инструкция по запуску

Откройте файл `index.html` в браузере(при этом предварительно загрузив `style.css` и `script.js`)


## Технологии

- HTML5
- CSS3 (flexbox, absolute positioning, keyframes)
- JavaScript (ES6+, DOM manipulation, events)
